# Рецензия тестов к книге «Приношение современному монашеству»

Пакет создаётся поэтапно. Первый редакционный файл охватывает главы 1–5:
`content_tests/drafts/ignatij_prinoshenie_01_05.json` (15 вопросов).

Тесты пока не встроены в книгу и не входят в release assets. Публикация
допустима после полного покрытия книги, автоматической проверки и человеческой
богословской рецензии.

## Источниковая основа

- [Полный текст на «Азбуке веры»](https://azbyka.ru/otechnik/Ignatij_Brjanchaninov/prinoshenie-sovremennomu-monashestvu/)
- Полное собрание творений и писем святителя Игнатия Брянчанинова. Том 4.
  «Приношение современному монашеству». Москва: Паломникъ, 2014. С. 7–394.

Нумерация в JSON учитывает композицию электронной книги:

1. «Правила наружного поведения для новоначальных иноков»;
2. введение к «Советам относительно душевного иноческого делания»;
3–5. первые три главы «Советов».

## Особое внимание рецензента

1. Монашеские предписания нельзя без оговорки переносить на мирян.
2. Послушание нельзя истолковывать как отмену совести, ответственности или
   евангельского рассуждения.
3. Внешний порядок у святителя служит внутреннему деланию и не заменяет его.
4. Упоминание суда не должно заслонять открытый покаянием путь возвращения.
5. Обещание явления Христа исполняющему заповеди относится к духовному
   присутствию Господа, а не к поиску чувственных видений.

## Формат вердикта

Для каждого вопроса следует указать: `принять`, `исправить` или `снять`, а
замечание прислать текстом в теле сообщения. Вложения с рецензиями могут
передаваться ненадёжно.
